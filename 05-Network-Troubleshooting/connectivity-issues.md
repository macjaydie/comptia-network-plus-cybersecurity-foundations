# Network Connectivity Issues

## Local Connectivity Problems

Examples:
- Faulty cables
- Disabled network adapters
- Incorrect IP configuration

---

## Remote Connectivity Problems

Examples:
- Routing failures
- ISP outages
- Firewall blocks

---

## Troubleshooting Approach

- Verify local network access
- Test gateway connectivity
- Test external connectivity
- Identify where communication fails

---

## SOC Relevance

Connectivity issues may indicate:
- Network outages
- Firewall blocks
- DDoS conditions
