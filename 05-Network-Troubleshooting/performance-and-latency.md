# Network Performance and Latency Issues

## Symptoms of Performance Problems

- Slow network speeds
- High latency
- Packet loss
- Application timeouts

---

## Common Causes

- Network congestion
- Bandwidth saturation
- Faulty hardware
- Misconfigured QoS

---

## Diagnostic Tools

- ping (latency and loss)
- tracert (hop delays)
- netstat (connection volume)

---

## Security Considerations

Performance degradation may indicate:
- DDoS attacks
- Network scanning
- Malicious traffic floods
