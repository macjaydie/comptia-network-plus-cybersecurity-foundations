# Common Network Issues

## IP Address Conflicts

Occurs when two devices share the same IP address.

Symptoms:
- Intermittent connectivity
- Network access errors

---

## DNS Issues

Symptoms:
- Websites not loading by name
- IP access works but domains fail

---

## DHCP Failures

Symptoms:
- No IP address assigned
- APIPA address (169.254.x.x)

---

## Firewall Misconfigurations

Symptoms:
- Blocked legitimate traffic
- Inaccessible services

---

## Security Context

SOC analysts must differentiate between:
- Misconfigurations
- Malicious activity
