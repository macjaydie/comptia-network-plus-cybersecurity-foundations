# VPN and Remote Access

## What is a VPN?

A Virtual Private Network (VPN) encrypts network traffic and creates a secure tunnel over an untrusted network.

---

## Types of VPNs

### Site-to-Site VPN
Connects entire networks together.

### Remote Access VPN
Allows individual users to securely connect to a network.

---

## VPN Protocols

- IPsec
- SSL/TLS
- L2TP

---

## Security Considerations

- Weak credentials can compromise VPN access
- Split tunneling increases risk
- VPN logs are critical for incident investigations

SOC analysts monitor VPN authentication and session logs.
